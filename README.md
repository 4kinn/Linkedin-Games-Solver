This repository contains JavaScript-based "Bookmarklets" to automatically solve LinkedIn's daily **Sudoku** and **Queens** games.

---

## üöÄ How to Install & Use

You don't need to install any extensions. We will simply use your browser's bookmarks bar.

1.  Make sure your **Bookmarks Bar** is visible in your browser (Chrome, Edge, etc.).
2.  Right-click on the bar and select **"Add Page"**.
3.  **Name:** Enter a name for the button (e.g., `Sudoku Solver`).
4.  **URL:** Copy the code block below (for the specific game) and paste it into the URL field.
5.  **Save** it.
6.  Go to the LinkedIn game page and click your new bookmark button!
